
```
400 > Bad Request
404 > Not Found > el servidor no encuentra el recurso
500 > Internal Server Error > error interno del servidor
200 > OK > todo esta bien
201 > Created > se creo un recurso
204 > No Content > no hay contenido
301 > Moved Permanently > el recurso se movio permanentemente
302 > Found > el recurso se movio temporalmente
304 > Not Modified > el recurso no ha sido modificado
```
